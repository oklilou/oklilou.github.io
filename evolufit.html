Evolufit

Site d'abonnement salle de sport

Résumé du projet

Dans le cadre de nos cours sur les frameworks JavaScript, nous avons utilisé React pour développer EvoluFit, un site d'abonnement dédié aux salles de sport. React s'est avéré être le choix idéal en raison de sa flexibilité et de son efficacité dans la construction d'interfaces utilisateur interactives.
Malgré ma familiarité limitée avec React au début, j'ai rapidement découvert ses avantages en termes de construction d'interfaces utilisateur réactives et modulaires. Cette expérience a été un tremplin d'apprentissage, me permettant de maîtriser les principes de React tout en développant une plateforme fonctionnelle et intuitive pour EvoluFit.

Le projet

En intégrant du JavaScript directement dans le code HTML, nous permettons à nos pages web d'interagir dynamiquement avec les utilisateurs. Plutôt que de charger l'intégralité du code JavaScript directement dans le fichier HTML, il est souvent plus efficace de séparer les fichiers et d'appeler des fonctions spécifiques. Cette approche non seulement améliore la lisibilité du code, mais simplifie également la maintenance et la gestion des fonctionnalités.
